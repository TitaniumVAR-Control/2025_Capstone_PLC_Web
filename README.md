# PLC 모니터링 웹 대시보드

VAR(Vacuum Arc Remelting) 공정 실시간 모니터링을 위한 웹 대시보드

## 기술 스택

- **React 18** + **TypeScript**
- **Vite** - 빌드 도구
- **Tailwind CSS** - 스타일링
- **Recharts** - 차트 라이브러리
